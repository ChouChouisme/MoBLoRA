# MoBLoRA 🎯

**"From Experts to Bases: Orthogonal Subspace Mixture for Continual Multimodal Instruction Tuning"**

✨ ACL 2026 Main Conference

## 📖 Introduction

MoBLoRA is a parameter-efficient framework for Multimodal Continual Instruction Tuning (MCIT). By decomposing LoRA weights into a globally shared orthonormal basis pool and task-specific mixing matrices, MoBLoRA achieves zero forgetting on the CoIN benchmark while outperforming all existing methods.
